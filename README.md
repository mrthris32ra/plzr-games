# PlezierGames


### 🏷️ History
PlezierGames is an interactive website that contains many pages, declared from the dutch word *plezier* meaning fun, and **games** such as video games, tools and utilities. In the late 2024, Mr. Thristan Ezra, a Javascript Programmer & Game Developer, just find out 
that he read every dutch, greek, and latin dictionary to create the name of his website. 💡✏️

On 2025, Mr. Thristan Ezra begin creating his website in 3 html pages which includes update log, games, and third-party social links that are added to the website. The website has maximum of 16 games with ongoing updates. 🔍
